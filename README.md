此项目使用Flask制作的用于可视化本地通过ssh远程访问修改云端服务器文档内容。S

使用方法：
-cd SSHdocumentuploader\frontend\src\
-pip install -r requirements.txt
-修改app.py中服务器ip以及账户密码，以及启用，禁用地址。
-在src中运行app.py
